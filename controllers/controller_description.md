# Explanation for the controllers in this repo

## Pursuer Scripts:
- basic_tracker.py 
    - simple pd control with no distance bound, no obstacle avoidance

- mpc_noa.py
    - non-occlusion aware mpc script
    - made to demonstrate the case of not considering occlusion 


## Evader Scripts: 
- manual_control.py
    - allows w,a,s,d control of the evader manually by human input

- scripted_evader_1.py 
    - to be used along with single_circle.py environment
